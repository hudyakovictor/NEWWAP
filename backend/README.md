# NewApp Backend

Бекенд новой версии проекта.

Запуск:

```bash
cd /Users/victorkhudyakov/dutin/newapp/backend
uvicorn main:app --reload --host 127.0.0.1 --port 8011
```

Ключевые возможности:

- сканирование основного и калибровочного наборов;
- реальная pose-классификация через `app/detect_pose.py`;
- реальная 3D-реконструкция и UV extraction через старые ядра;
- сохранение артефактов по каждой фотографии в `newapp/storage`;
- хронология строго внутри одного ракурса;
- статистика калибровки и рекомендации по закрытию белых зон.
